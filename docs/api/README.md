# API Overview

## Role of API
API є внутрішнім (internal) контрактом для frontend застосунку.

## Usage
API використовується SPA клієнтом (web frontend).

## Architecture Style
API побудований за REST принципами і працює через HTTP + JSON.

## Main Features
- управління замовленнями
- отримання даних
- створення та видалення ресурсів
