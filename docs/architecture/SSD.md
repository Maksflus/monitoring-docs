# System Specification Document (SSD)

## 1. Purpose
Система призначена для моніторингу логів та метрик.

## 2. Functional Requirements

- FR-01: Збір логів
- FR-02: Аналіз метрик
- FR-03: Нотифікації

## 3. Non-functional Requirements

- Performance: до 200 ms
- Availability: 99.9%
- Security: JWT
