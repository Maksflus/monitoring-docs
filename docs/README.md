# Monitoring System Documentation

## Overview
Система моніторингу логів та метрик призначена для збору, аналізу та візуалізації даних у реальному часі. Вона дозволяє відстежувати стан сервісів, реагувати на інциденти та забезпечувати стабільність інфраструктури.
